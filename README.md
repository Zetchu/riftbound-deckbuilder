# Riftbound Decksmith

## Project Description

Riftbound Decksmith is a specialized deck-building and analytics web application for the newly released _Riftbound TCG_. It provides players with a clean, fast interface to browse the entire card database, theory-craft new decks, and validate them against official tournament rules.

Built with scalability in mind, this project aims to become a central hub for the competitive Riftbound community, helping players make data-driven decisions about their decks.

## Potential Future Features

- **Advanced Card Filtering:** Sort the database by cost, faction, rarity, and specific card mechanics.
- **Interactive Deck Builder:** Drag-and-drop UI to easily add/remove cards while tracking the 40-card limit.
- **Analytics Dashboard:** Visual charts (Mana Curve, Faction Distribution, Card Type breakdown) for the active deck.
- **Import/Export Engine:** Generate and parse clipboard codes so users can easily share decks on Reddit or Discord.
- **Collection Tracker:** A tool for users to input the physical/digital cards they own to calculate how close they are to building a top-tier meta deck.
